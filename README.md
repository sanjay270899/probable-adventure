# Devsnest's Frontend

## How to run

1. Make sure you have installed [pnpm](https://pnpm.io/installation)

2. Install the dependencies

```bash
pnpm i
```

3. Start the dev server

```bash
pnpm dev
```

This will start the project at http://localhost:3000/
